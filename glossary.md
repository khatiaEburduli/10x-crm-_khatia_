# Glossary

## CRUD
Create, Read, Update and Delete operations used to manage CRM clients.

## localStorage
Browser storage used to save users, clients, session and theme data.

## Authentication
The process of verifying a user's identity during login.

## Authorization
Restricting access to protected pages for unauthorized users.

## Session
Information about the currently logged-in user stored in crm_session.

## API
Application Programming Interface used to communicate with DummyJSON.

## Fetch API
JavaScript API used to send HTTP requests.

## Async/Await
JavaScript syntax for handling asynchronous operations.

## Validation
Checking user input before processing or saving data.

## Toast Notification
A temporary message displayed to inform the user about successful or failed actions.

## DOM
Document Object Model used to manipulate HTML elements with JavaScript.
